# Random User Generator
This project consumes an webAPI and generates a person with respective credentials, such as:

* name & gender
* Id
* Mail
* Phone number
* Country
* Random job

Random user API: https://randomuser.me/

## Project built with default React-Native CLI :

* Verion : 0.72.3
